## 📋 Tentang

Proyek **Fake Update** ini adalah simulasi layar **Windows 11 Update** palsu yang bisa dipakai untuk **prank** teman atau keluarga. Cukup buka halaman ini di browser korban, atur durasi, lalu layar akan berubah menjadi tampilan update Windows 11 palsu — lengkap dengan animasi loading dan progress百分比 yang berjalan otomatis.

Mode *fullscreen* aktif secara otomatis, dan keluar dari fullscreen akan mengembalikan panel setup.

## 🚀 Cara Pakai

1. **Clone repositori ini**
   ```bash
   git clone https://github.com/username/fake-update.git
   cd fake-update
   ```

2. **Buka `index.html`** di browser (langsung double-click atau lewat live server)

3. **Atur panel setup**:
   - **Update time** — berapa menit durasi "update" palsu
   - **Start time** — persentase awal (biar makin meyakinkan)

4. **Klik "Mulai Prank"** dan lihat reaksi korban! 🎭

> **Tips:** Hosting di **GitHub Pages** biar bisa diakses dari HP korban via link!
